# Skyline
The codebase for my master thesis. It follows a naming scheme of cloud formations and the sky.

## Theme: Cloud formations
Possible names regarding cloud formations
1. Cirrus:
   - Feathers
   - HaloEffects
   - JetStrams
2. Cumulus:
  - PuffyFeatures
  - Daydreams
  - SkyVistas
3. Stratus
  - Overlays
  - Mists
  - RainDrizzles
4. Nimbus
  - Downpours
  - Snowflakes
  - Thunderbolts
5. Altocumulus or Altostratus
  - MixedBags
  - HorizonLines
  - SkyPatterns
